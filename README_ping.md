## Descripción breve del módulo.
Desarrollo de un modulo en python que compruebe que la IP sea válida y luego haga ping, sin mostrar en pantalla el comando.

## Sintaxis de uso.
Para poder ejecutarlo será de la siguiente manera:
```bash
python comprobacionping.py dirección_IP

```

## Requisitos
### Software
- Python 3.10 o superior

### Herramientas necesarias
- Terminal de comandos para ejecutar el script

### A tener en cuenta
- Ejecutar el script desde la carpeta donde se encuentre el archivo `.py` o usar la ruta hasta el archivo
